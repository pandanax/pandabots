Потенциальные улучшения
🔴 Критично
Лимит Telegram 4096 символов - в коде есть предупреждение, но нет технической защиты. Рекомендую добавить проверку длины и разбивку длинных сообщений.
Max tokens = 1000 - может быть недостаточно для развернутых ответов. DeepSeek может обрезать ответ на полуслове.
История ограничена 10 сообщениями - для длинных консультаций может не хватить контекста.
🟡 Желательно
Нет обработки команд - /start, /help обрабатываются как обычные сообщения
Нет rate limiting - пользователь может спамить запросами к DeepSeek API
Markdown парсинг хрупкий - в промпте ограничения по синтаксису, но AI может их нарушить
